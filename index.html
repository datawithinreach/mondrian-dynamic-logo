Generative Identity

A dynamic logo design project inspired by Piet Mondrian's neoplasticism. Consisting of shifting structures representing fluid data patterns and connections brought within reach.

Option A (Rectangular)

Standard rectangular grid structure containing dynamic color-block shifts.

Option B (Rhombus Logo)

A diamond geometric composition containing horizontal and vertical grids. Clean, modern geometric badge style.

Option C (Circular Logo)

A circular badge style where neoplastic cells are constrained inside a perfect loop. Extremely premium visual identity.

Usage Guide

  • Click Next Layout on any version to trigger animated grid shifts.
  • Enable Auto Play to run them as ambient screen savers.
  • Export as SVG for papers (vector format) or high-res **PNG** for slides.
